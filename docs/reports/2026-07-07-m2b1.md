## ⚠️ 전제 (Firebase 보안 규칙)
이 앱은 **인증 사용자에게 `cs/*` 읽기 + `cs/config` 쓰기 권한**이 있어야 동작(HK의 `app/*`와 동일 가정). 규칙에 `cs/*`가 없으면 로그인 후 "읽기 권한 오류" 토스트가 뜨고 카드가 비어 있음 → 그 경우 규칙에 `cs/*` 허용 추가 필요(회신 요망).

## 클라라 확인 절차
1. **gas/Code.gs 재반영**(붙여넣기) — 다음 `pollCsInbox` run부터 `cs/meta/gmailBudget` 기록 시작.
2. `https://pwr-clair.github.io/cs/` 접속 → 로그인 화면 → **HK 계정(이름+비밀번호, 대소문자 무관)**으로 로그인.
3. `cs/drafts`가 비어 있으면 **"대기 중인 초안 없음"** 정상. 초안이 생기면 대기 카드로 표시.
4. **autoSend 토글** → `cs/config/autoSend` 저장(발송 영향 0, M2b-2에서 연동). 예산 게이지는 미러 데이터가 들어오면 표시.
5. 승인/수정/발송 버튼이 **비활성(회색)** 인지 확인(읽기 전용).
