## 미확정/가정 (Fable 확인 요청)
1. **pendingBookings 실경로**: `pendingBookings` → `app/pendingBookings` 순으로 시도. 둘 다 아니면 실경로 회신 요망.
2. **room 필드명**: `room|roomName|unit|roomType` 순 시도 → 없으면 푸시에 "미상". 실제 방 필드명 알려주면 정확 표기.
3. **CS-DB 시트 열**: 헤더에 상황/요약/질문/문의 · 답변/응대/회신 · 언어/lang 키워드로 매핑. 매핑 실패 시 로그에 헤더 출력 → 열 이름 회신 요망.
4. 유사사례 검색은 임베딩 아님(키워드 겹침) — CJK는 토큰 겹침이 약해 언어 매칭 위주. 정교화는 후속.
5. category는 자유 분류(한국어 짧은 라벨), confidence는 저장만·게이팅 미사용(설계 준수).
